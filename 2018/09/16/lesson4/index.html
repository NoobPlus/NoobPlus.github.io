empty(var) — 检查一个变量是否为空

如果 var 是非空或非零的值，则 empty() 返回 FALSE
换句话 说， “” 、 0 、 0.0 、 “0” 、 NULL 、 FALSE 、 array();
以及没有任何属性的对象都将被认为是空的，如果 var 为空，则返回 TRUE
